## MCP Capabilities

<Info>
This webhook endpoint is enabled for Model Context Protocol (MCP), allowing AI assistants to understand and help configure webhook integrations.
</Info>

### What MCP enables

With MCP enabled for this webhook, AI assistants can:

- **Explain webhook behavior**: Describe when and how this webhook is triggered
- **Generate webhook handlers**: Create code to receive and process webhook events
- **Validate payloads**: Help verify incoming webhook data matches the expected schema
- **Set up integrations**: Guide users through configuring webhook endpoints

### Example AI interactions

Users can ask:
- "What data does the plant webhook send?"
- "How do I set up a webhook handler for new plants?"
- "Show me a Node.js server that receives this webhook"

The AI assistant uses MCP to:
1. Understand the webhook payload structure
2. Generate appropriate handler code
3. Provide best practices for webhook security and validation
4. Help troubleshoot webhook delivery issues
