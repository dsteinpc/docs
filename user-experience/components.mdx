To assemble the user experience patterns shown on the previous page, we use a mix of standard and custom components. Whenever possible, we default to existing components from the Core design system, including these atomic components:

- [Card](https://core.procore.com/12.29.0/web/components/card/react/)
- [Button](https://core.procore.com/12.29.0/web/components/dropdown/react/)
- [Dropdown](https://core.procore.com/12.29.0/web/components/dropdown/react/)
