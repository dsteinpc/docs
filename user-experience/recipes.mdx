Below are the six UX recipes we've established in order to enable a "good" cross-account collaboration experience. **You are not required to use all six recipes for your own experience, but can pick and choose as necessary.**


Review each recipe for it's job-to-be-done, do's-and-don'ts, and resources to see it in action or start using it yourself.

## Send Copies of Internal Data

<Callout icon="user" color="#FFC107" iconType="regular">
**Job-to-be-Done:** I need to proactively communicate information with my collaborators.
</Callout>

<Tabs>
<Tab title="Web"><Frame>
  ![Send RFI to Another Account](/images/docs-images/UX-Pattern-01.png)
</Frame></Tab>
<Tab title="Mobile"><Frame>
  ![Send RFI to Another Account](/images/docs-images/UX-Pattern-01.png)
</Frame></Tab>
</Tabs>

#### Basic Journey
1. User identifies what they want to share
2. User uses **Export > Send to...** to select account with which they want to share
3. User confirms that item **cannot be un-sent**
4. User sees indicator that an item has been shared with an external collaborator
